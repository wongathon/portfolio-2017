Portfolio

Schedulister

Schedulister is a smart to-do list app that handles task display that helps users manage their habits. Displays daily tasks based on the current date, using backend logic to recalculate when tasks will next appear, handling daily, weekly, and monthy recurring tasks as well as one-time scheduled tasks. Full MERN stack app with JS and Moment.js logic.

Roadtrip Playlister

Roadtrip Playlister was my first project app, built with Javascript/HTML/CSS, using jQuery and APIs to generate a playlist based on geographic points along a mapped route, considering the places the user will pass on the route and how long the trip will take. The idea was to give you a thematic playlist for any roadtrip!

Tattoo Zone

Tattoo-Zone is meant to be a study in creating a social media app, focusing on user/post/comment database associations popular in common social media sites. The app also allows posting and uploading of pictures, commenting on others' posts, and logging in/loggin out states. We use jQuery and Sequelize, none of those fancy Node/React packages!
